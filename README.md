# taiki_osaka_graph

A Web/Code project by [Code for FUKUI](https://github.com/code4fukui).

**Live demo**: https://code4fukui.github.io/taiki_osaka_graph/

## Features

- Data visualization
- Canvas-based rendering
- CSV data parsing

## Technology Stack

**Browser modules (ES imports):**

- [`CSV.js`](https://js.sabae.cc/CSV.js)
- [`Chart.mjs`](https://js.sabae.cc/Chart.mjs)


## Installation & Setup

No build step required. Clone the repository and open `index.html` in a browser,
or visit the live demo at https://code4fukui.github.io/taiki_osaka_graph/
