# taiki_osaka_graph
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Visualizes the transition of oxidants (OX) in Osaka Prefecture, Japan.

## Demo
https://code4fukui.github.io/taiki_osaka_graph/

## Features
- Displays a line chart of OX levels over time
- Uses open data from the Osaka Prefectural Government

## Data / API
The data is fetched from the [Open Data of Osaka Prefecture Air Pollution Monitoring](https://taiki.kankyo.pref.osaka.jp/taikikanshi/) website.
