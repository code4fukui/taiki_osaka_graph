# taiki_osaka_graph

日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

大阪府における酸化物(OX)の推移を可視化します。

以下は英語のマークダウンチャンクを日本語に翻訳したものです。

## デモ
https://code4fukui.github.io/taiki_osaka_graph/

## 機能
- 時間の経過とともにOXレベルの推移を示す折れ線グラフを表示します
- 大阪府の公開データを使用しています

## データ / API
データは[大阪府の大気汚染監視データ(オープンデータ)](https://taiki.kankyo.pref.osaka.jp/taikikanshi/)のウェブサイトから取得されます。
